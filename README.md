# Exploring the Impact of AI Technologies Across Sectors
This project provides a visual exploration of how AI technologies are used across different industries based on dataset trends.  
Python + Matplotlib + Seaborn were used to analyze category frequencies and visualize variations across sectors and tools.

## Key Visualizations
- Distribution of AI-related topics across sectors
- Most referenced AI tools and technologies
- Comparison of sector-wise engagement
- Category frequency patterns

## Contents
- `notebook.ipynb` — all visualizations and analysis
- `Global_AI_Content_Impact_Dataset.csv` — dataset used
- `screenshots/` — exported charts for quick viewing

## Tools Used
Python, Matplotlib, Seaborn, Pandas
